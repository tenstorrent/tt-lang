# Security Policy

## Supported Versions

Currently, tt-lang is in active development. Security updates will be provided for the latest version.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing:
security@tenstorrent.com

Please do not open public issues for security vulnerabilities.

We will respond within 48 hours and work with you to address the issue.

## Security Update Process

1. Vulnerability is reported via email
2. Tenstorrent team acknowledges receipt within 48 hours
3. Team investigates and determines severity
4. Fix is developed and tested
5. Security advisory is published with fix details
6. Patch is released

Thank you for helping keep tt-lang and its users secure.
